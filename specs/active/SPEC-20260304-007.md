# SPEC-20260304-007: CLI Pagination Fixes

## Status: Active
## Priority: P0

## Objective
Fix two CLI commands that crash with `ArgumentValidationError: missing paginationOpts`.

## Module Overview
- `agentforge logs` calls `api.logs.list` (paginated) without passing `paginationOpts`
- `agentforge threads inspect <id>` calls `api.messages.list` (paginated) without `paginationOpts`
- Both commands need to pass `{ cursor: null, numItems: 50 }` as paginationOpts

## Root Cause
Convex paginated queries require `paginationOpts: { cursor: null | string, numItems: number }`.
The CLI passes `{}` or `{ threadId }` without including the required `paginationOpts` field.
