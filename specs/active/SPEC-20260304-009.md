# SPEC-20260304-009: Add threads rename Command

## Status: Active
## Priority: P1

## Objective
`agentforge threads rename <id> <new-name>` is missing from the CLI.
The Convex mutation `api.threads.rename` (or similar) should already exist or needs creating.

## Success Criteria
- [ ] `agentforge threads rename <threadId> "New Name"` renames the thread in Convex
- [ ] Short Convex `_id` accepted (not just the full internal ID)
- [ ] Success message printed: `Thread renamed to "New Name"`
- [ ] Error if thread not found
- [ ] `agentforge threads --help` shows the `rename` subcommand
- [ ] `agentforge threads list` shows updated name after rename
- [ ] Convex mutation for thread rename implemented if missing
- [ ] CLI dist rebuilt
- [ ] All 911+ tests pass
- [ ] Zero TypeScript errors

## Technical Details
- Add `threads rename <id> <name>` subcommand in `packages/cli/src/commands/threads.ts`
- Check if Convex has a `threads.rename` mutation in `convex/threads.ts`; if not, add:
  ```typescript
  export const rename = mutation({
    args: { id: v.id("threads"), name: v.string() },
    handler: async (ctx, args) => {
      await ctx.db.patch(args.id, { name: args.name });
    },
  });
  ```
- Support short ID lookup: try `ctx.db.get(id)` first, then fall back to index
