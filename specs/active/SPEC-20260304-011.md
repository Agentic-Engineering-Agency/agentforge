# SPEC-20260304-011: Wire Token Usage Logging (inputTokens / outputTokens / cost)

**Status:** Active  
**Priority:** P0 — Usage & Observability pages show all zeros  
**Branch:** feat/overnite-b  
**Worktree:** /tmp/agentforge-overnite-b  

## Problem

The Usage page and Observability page in the dashboard both show zeros because `mastraIntegration:executeAgent` never writes token counts to the logs table. The Mastra AI SDK returns usage data (`inputTokens`, `outputTokens`) but we discard it.

## Root Cause

In `convex/mastraIntegration.ts`, the `executeAgent` action:
1. Calls `agent.generate()` → gets back `{ text, usage: { inputTokens, outputTokens } }`
2. Records a log entry but doesn't include token/cost fields
3. The `logs` table schema may not have these fields yet

## Acceptance Criteria

### Convex Schema
```typescript
// convex/schema.ts — logs table must have:
logs: defineTable({
  agentId: v.string(),
  sessionId: v.optional(v.string()),
  threadId: v.optional(v.string()),
  level: v.string(),           // 'info' | 'error' | 'warning'
  message: v.string(),
  metadata: v.optional(v.any()),
  // NEW fields:
  inputTokens: v.optional(v.float64()),
  outputTokens: v.optional(v.float64()),
  totalTokens: v.optional(v.float64()),
  costUsd: v.optional(v.float64()),
  model: v.optional(v.string()),
  provider: v.optional(v.string()),
}).index("byAgent", ["agentId"])
 .index("byCreationTime", ["_creationTime"]),
```

### mastraIntegration.ts
After `agent.generate()`, extract usage and write to logs:
```typescript
const result = await agent.generate(messages);
const usage = result.usage ?? {};
await ctx.runMutation(internal.logs.add, {
  agentId,
  threadId,
  sessionId,
  level: 'info',
  message: `Agent response: ${result.text?.slice(0, 100)}`,
  inputTokens: usage.inputTokens ?? 0,
  outputTokens: usage.outputTokens ?? 0,
  totalTokens: (usage.inputTokens ?? 0) + (usage.outputTokens ?? 0),
  // Estimate cost (OpenAI gpt-4o-mini: $0.15/1M input, $0.60/1M output)
  costUsd: ((usage.inputTokens ?? 0) * 0.00000015) + ((usage.outputTokens ?? 0) * 0.0000006),
  model: agentConfig.modelId,
  provider: agentConfig.providerId,
});
```

### analytics.ts / usage.ts
Update `getUsageSummary` to return:
- `totalInputTokens`, `totalOutputTokens`, `totalTokens`
- `totalCostUsd`
- `byAgent`: per-agent breakdown

### Dashboard: usage.tsx

### Dashboard: observability.tsx
